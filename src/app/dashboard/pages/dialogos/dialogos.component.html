Metodo open

Vamos a abrir una ventana emergente que contiene el componente dado. El método recibe el componente y un objeto de configuración opcional.

  let dialogRef = dialog.open(UserProfileComponent, {
    height: '400px',
    width: '600px',
  });

El método open retorna una instancia de MatDialogRef

El objeto de configuracion tiene las siguientes propiedades:

PASO 3: ABRIR EL COMPONENTE DIALOGO

Usamos el servicio, el componente que se mostrara y un objeto de configuración

Clase del componente padre Plantilla del componente padre 
import {Component, inject} from '@angular/core';
import {MatDialog} from "@angular/material/dialog";
import {HolaMundoComponent} from "../../dialogs/hola-mundo/hola-mundo.component";


@Component({
  selector: 'app-dialogos',
  standalone: true,
  imports: [HolaMundoComponent],
  templateUrl: './dialogos.component.html',
  styleUrl: './dialogos.component.scss'
})

export class DialogosComponent {
  readonly dialog = inject(MatDialog);
  
  openDialog(){
    let dialogRef = this.dialog.open(HolaMundoComponent, {
      height: '400px',
      width: '600px',
    })
  }
  
}
<button (click)="openDialog()" >Abrir</button >
